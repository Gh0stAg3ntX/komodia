00011 //    This file contains the routines necessary to implement the WSPAsyncSelect
00012 //    API. In order for our LSP to count the bytes sent and received by an 
00013 //    application when the app uses WSAAsyncSelect we need to intercept all
00014 //    of the app's send and receive calls. To do this we implement a hidden
00015 //    window. When the app makes a send/recv call we intercept this and post
00016 //    the operation with the provider's socket to our own window. This allows
00017 //    the LSP to receive the completion notification. At this point we will

00025 //    occured. If the LSP doesn't capture this then the wrong socket handle
