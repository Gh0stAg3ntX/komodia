This VB.Net sample interfaces with the "SSLHijackClientCOM.dll" which is a COM wrapper for the main SDK DLL: "SSLHijackClientDLL.dll", this sample is intended to help you evaluate how to interface with the SDK before you requested it from Komodia, keep in mind that without the SDK this is just code, it will compile but will not do anything since it requires the SDK which is available only by requesting it from us.

How to work with the sample

This is the sample main screen:

Set the DLL

  1. Put the complete path to the main SDK DLL ("SSLHijackClientDLL.dll") in the "DLL Path" text box.
  2. Load the DLL according to the desired mode.
    1. Press: "Load DLL" to run the sample without using the Komodia's Interceptor HTTP Decoder.
    2. Press: "Load DLL HTTP" to run the sample using the Komodia's Interceptor HTTP Decoder (if you got a sample without the parser, then don't use this button because it will not work)

Wait for results

You should see the traffic on the main list box.
