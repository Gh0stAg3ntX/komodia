This product has been discontinued because the Komodia's Redirector is able to perform all of the functions the Interceptor had, the documents is online for clients who still need it for their version.

This page covers some aspects of Komodia's Interceptor that are more technical and does not fit in the promotional web page.

Komodia's Interceptor installation guide

There are two guides available:

  • Visit the page for Komodia's Interceptor installation guide, this is a complete guide.
  • Visit the page for Komodia's Interceptor quick installation guide, this is a quick step by step without going into details of why and how.

Samples

There are three samples we provide to work with:

  1. Interceptor C sample - To work with the SDK without HTTP Decoder.
  2. Interceptor C sample - To work with the SDK with HTTP Decoder.
  3. Interceptor VB.Net sample - To work with the SDK with or without HTTP Decoder.
